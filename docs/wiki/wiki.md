# Wiki Guardiãs D'Águas

## Requisitos para rodar

1. Node.js Versão 24.11.0
1. npm Versão 11.6.2

## Clonar repósitorio

1. Clone o repositório localmente:

```bash
git clone https://github.com/ProjetoIntegrador-2Semestre/GardiasDasAguas.git
```

2. Entre no reposítorio e rode o comando para instalar as dependencias do projeto:

```bash
npm install 
```

3. Quando finalizado a instalação das dependencias, poderemos rodar a parte do frontend localmente, como o comando:
OBS: você tem que estar no diretório /app

```bash
npm run dev
```
